This project focuses on analyzing customer/user behavior over time using cohort analysis and retention metrics to uncover trends in engagement, loyalty, and performance. The dashboard provides actionable insights into how different customer groups evolve after their first interaction, enabling data-driven decision-making for business growth and strategy optimization.

The solution was built using Power BI, incorporating advanced data modeling, DAX measures, and time-based cohort segmentation to evaluate retention patterns and performance across multiple periods.

**Key Objectives**
- Measure customer retention rate across different cohorts
- Identify drop-off trends and behavioral patterns over time
- Compare performance between cohorts to evaluate growth sustainability
- Support strategic decisions related to marketing, acquisition, and customer lifecycle management

**Features & Insights**
- Cohort segmentation based on first activity period
- Retention rate heatmaps for time-series analysis
- Performance tracking across multiple intervals
- Interactive filtering for dynamic exploration
- Clean and intuitive business-focused visualization design

**Tools & Technologies**
- Power BI
- Data Modeling (Star Schema Concepts)
- DAX Calculations
- Time Intelligence Functions
- Data Visualization & Storytelling
